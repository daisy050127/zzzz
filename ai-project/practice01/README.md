# LLM 客户端工具

## 功能用途

`llm_client.py` 是一个简单的 LLM（大型语言模型）客户端工具，主要功能包括：

1. **环境变量加载**：从 `.env` 文件中读取配置信息，如 API 密钥、模型名称、基础 URL 等
2. **Token 计算**：提供简化版的 token 计数功能
3. **LLM API 调用**：发送请求到指定的 LLM API（支持 OpenAI API 或本地 LM Studio）
4. **响应处理**：解析 API 响应并提取关键信息
5. **性能统计**：计算请求耗时、token 消耗和处理速度
6. **错误处理**：提供基本的错误捕获和提示

## 实现的教学目标

通过本代码的学习，可以实现以下教学目标：

1. **HTTP/HTTPS 请求**：学习如何使用 Python 的 `http.client` 模块发送 POST 请求
2. **JSON 处理**：学习如何构建 JSON 请求数据和解析 JSON 响应
3. **环境变量管理**：学习如何从 `.env` 文件加载配置信息
4. **URL 解析**：学习如何使用 `urllib.parse` 解析 URL 结构
5. **异常处理**：学习如何捕获和处理常见的异常情况
6. **性能分析**：学习如何计算和分析 API 调用的性能指标
7. **LLM 集成**：了解如何与 LLM API 进行交互的基本流程

## 使用方法

1. 在项目根目录创建 `.env` 文件，配置以下环境变量：
   - `API_KEY`：LLM API 的访问密钥
   - `BASE_URL`：LLM API 的基础 URL（默认为 `https://api.openai.com/v1`）
   - `MODEL`：使用的模型名称（默认为 `gpt-3.5-turbo`）
   - `TIMEOUT`：请求超时时间（默认为 30 秒）

2. 运行脚本：
   ```bash
   python llm_client.py
   ```

3. 脚本会发送一个测试请求，并输出回复内容和统计信息。
